## Word Tips
### Quick Navigation
- **Use `CTRL + [Arrow Key]`** to navigate word by word.
- **Use `CTRL + SHIFT + [Arrow Key]`** to select text word by word.

### Format Painter
- **Double-click the Format Painter** to apply formatting to multiple places.
